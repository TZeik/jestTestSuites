## 🔧 **Jest Test Suites**

# ☝️ **Para que sirven el Testing?**

*Testing es el proceso sistemático de ejecutar una aplicación con la intención de encontrar errores, defectos o comportamientos no deseados. Sirve para garantizar que el código funcione correctamente y cumpla los requisitos funcionales y no funcionales.*


Para utilización para de las test con Jest utilizar

$npm test 

o

$npm test <ruta del test>
